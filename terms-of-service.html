Deskripsi Layanan

FiveM Player Finder adalah bot Discord yang menyediakan informasi mengenai pemain dan status server pada game FiveM (modifikasi multiplayer untuk Grand Theft Auto V), dengan mengambil data dari API publik penyedia layanan FiveM (cfx-services.net). Bot ditawarkan secara gratis dengan fitur terbatas, dan/atau melalui skema berbayar (subscription) untuk akses fitur tambahan.
