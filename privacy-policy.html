Kebijakan Privasi

terakhir_diperbarui: [isi tanggal] · berlaku_untuk: FiveM Player Finder

Kebijakan Privasi ini menjelaskan bagaimana FiveM Player Finder ("Bot", "kami") mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan Bot di Discord.

01

Data yang Kami Kumpulkan

Saat Anda menggunakan command Bot (misalnya !cari), kami dapat mengumpulkan data berikut:

  • Username dan ID Discord Anda, untuk keperluan pencatatan log penggunaan (misalnya melacak batas penggunaan/limit fitur dan status subscription).
  • ID Server (Guild) Discord tempat Bot digunakan, untuk keperluan konfigurasi dan pengelolaan subscription per server.
  • Parameter command yang Anda kirim (misalnya nama server FiveM yang ingin dicari), digunakan secara real-time untuk memproses permintaan Anda.
Kami tidak menyimpan isi pesan (message content) Anda secara permanen. Bot hanya membaca pesan untuk mendeteksi command (sistem prefix command) dan tidak menyimpan teks pesan tersebut di luar proses itu.
02

Bagaimana Kami Menggunakan Data

Data yang dikumpulkan digunakan untuk:

  • Memproses dan menjalankan command yang Anda minta.
  • Mencatat riwayat penggunaan Bot untuk keperluan analitik internal dan pengelolaan subscription/limit.
  • Memberikan dukungan (support) jika Anda mengalami masalah dengan Bot.
  • Meningkatkan kualitas dan performa Bot.
03

Data dari Pihak Ketiga

Untuk menampilkan informasi server dan pemain FiveM, Bot mengambil data secara real-time dari API publik milik Cfx.re/FiveM (cfx-services.net). Data ini bukan data pribadi pengguna Discord, melainkan data publik server game FiveM.

Jika Bot menyediakan fitur berbayar, proses pembayaran dapat melibatkan pihak ketiga seperti penyedia QRIS, Saweria, atau Trakteer. Kami hanya menerima konfirmasi status pembayaran dan tidak menyimpan data sensitif seperti nomor kartu/rekening Anda.

04

Penyimpanan & Keamanan Data

  • Data log penggunaan (username, ID Discord, ID server) disimpan di server/database milik kami dengan langkah-langkah keamanan yang wajar untuk mencegah akses tidak sah.
  • Data disimpan selama akun/server Anda masih aktif menggunakan Bot, atau sesuai kebutuhan untuk keperluan subscription.
05

Hak Pengguna

  • Anda dapat menghentikan pengumpulan data dengan tidak menggunakan command Bot, atau dengan mengeluarkan (kick) Bot dari server Discord Anda.
  • Anda dapat meminta penghapusan data log Anda dengan menghubungi kami melalui kontak di bawah.
06

Privasi Anak

Bot ini tidak ditujukan untuk anak-anak di bawah usia minimum yang ditetapkan oleh Discord Terms of Service (saat ini 13 tahun, atau usia minimum yang lebih tinggi di wilayah tertentu). Kami tidak secara sengaja mengumpulkan data dari anak di bawah usia tersebut.

07

Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan signifikan akan diumumkan melalui [isi: server Discord support / changelog bot].
