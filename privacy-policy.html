Kebijakan Privasi

terakhir_diperbarui: 15 June 2026 · berlaku_untuk: FiveM Player Finder

Kebijakan Privasi ini menjelaskan bagaimana FiveM Player Finder ("Bot", "kami") mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan Bot di Discord.

01

Data yang Kami Kumpulkan

Saat Anda menggunakan command Bot (misalnya !cari), kami dapat mengumpulkan data berikut:

  • Username dan ID Discord Anda, untuk keperluan pencatatan log penggunaan (misalnya melacak batas penggunaan/limit fitur dan status subscription).
  • ID Server (Guild) Discord tempat Bot digunakan, untuk keperluan konfigurasi dan pengelolaan subscription per server.
  • Parameter command yang Anda kirim (misalnya nama server FiveM yang ingin dicari), digunakan secara real-time untuk memproses permintaan Anda.
Kami tidak menyimpan isi pesan (message content) Anda secara permanen. Bot hanya membaca pesan untuk mendeteksi command (sistem prefix command) dan tidak menyimpan teks pesan tersebut di luar proses itu.
02

Bagaimana Kami Menggunakan Data

Data yang dikumpulkan digunakan untuk:

  • Memproses dan menjalankan command yang Anda minta.
  • Mencatat riwayat penggunaan Bot untuk keperluan analitik internal dan pengelolaan subscription/limit.
  • Memberikan dukungan (support) jika Anda mengalami masalah dengan Bot.
  • Meningkatkan kualitas dan performa Bot.
03

Data dari Pihak Ketiga

Untuk menampilkan informasi server dan pemain FiveM, Bot mengambil data secara real-time dari API Penyedia. Data ini bukan data pribadi pengguna Discord, melainkan data publik server game FiveM.

Jika Bot menyediakan fitur berbayar, proses pembayaran dapat melibatkan pihak ketiga seperti penyedia QRIS. Kami hanya menerima konfirmasi status pembayaran dan tidak menyimpan data sensitif seperti nomor kartu/rekening Anda.

04

Penyimpanan & Keamanan Data

  • Data log penggunaan (username, ID Discord, ID server) disimpan di server/database milik kami dengan langkah-langkah keamanan yang wajar untuk mencegah akses tidak sah.
  • Data disimpan selama akun/server Anda masih aktif menggunakan Bot, atau sesuai kebutuhan untuk keperluan subscription.
05

Hak Pengguna

  • Anda dapat menghentikan pengumpulan data dengan tidak menggunakan command Bot, atau dengan mengeluarkan (kick) Bot dari server Discord Anda.
  • Anda dapat meminta penghapusan data log Anda dengan menghubungi kami melalui kontak di bawah.
06

Privasi Anak

Bot ini tidak ditujukan untuk anak-anak di bawah usia minimum yang ditetapkan oleh Discord Terms of Service (saat ini 13 tahun, atau usia minimum yang lebih tinggi di wilayah tertentu).

07

Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan signifikan akan diumumkan melalui server Discord Support.

08

Kontak

Jika Anda memiliki pertanyaan mengenai Kebijakan Privasi ini atau ingin meminta penghapusan data, silakan hubungi kami melalui:

  • Server Discord Support: https://discord.gg/5TQTfSnSNp
  • Email: iqballoww@gmail.com

Privacy Policy

last_updated: 15 June 2026 · applies_to: FiveM Player Finder

This Privacy Policy explains how FiveM Player Finder ("Bot", "we") collects, uses, and protects your information when you use the Bot on Discord.

01

Data We Collect

When you use a Bot command (e.g. !cari), we may collect the following data:

  • Your Discord username and ID, for usage logging purposes (e.g. tracking feature usage limits and subscription status).
  • The Discord Server (Guild) ID where the Bot is used, for configuration and per-server subscription management.
  • Command parameters you send (e.g. the FiveM server name you want to search for), used in real time to process your request.
We do not permanently store your message content. The Bot only reads messages to detect commands (a prefix command system) and does not store the message text beyond that process.
02

How We Use Data

The data collected is used to:

  • Process and run the commands you request.
  • Log Bot usage history for internal analytics and subscription/limit management.
  • Provide support if you experience issues with the Bot.
  • Improve the quality and performance of the Bot.
03

Third-Party Data

To display FiveM server and player information, the Bot fetches data in real time from a provider API. This data is not Discord users' personal data, but public FiveM game server data.

If the Bot offers paid features, the payment process may involve third parties such as QRIS providers. We only receive payment status confirmation and do not store sensitive data such as your card or account numbers.

04

Data Storage & Security

  • Usage log data (username, Discord ID, server ID) is stored on our own servers/database with reasonable security measures to prevent unauthorized access.
  • Data is stored for as long as your account/server remains active with the Bot, or as needed for subscription purposes.
05

User Rights

  • You can stop data collection by not using Bot commands, or by removing (kicking) the Bot from your Discord server.
  • You can request deletion of your log data by contacting us using the details below.
06

Children's Privacy

This Bot is not intended for children under the minimum age set by Discord's Terms of Service (currently 13, or a higher minimum age in certain regions).

07

Changes to this Policy

We may update this Privacy Policy from time to time. Significant changes will be announced via our Discord support server.

08

Contact

If you have questions about this Privacy Policy or wish to request data deletion, please contact us via:

  • Discord Support Server: https://discord.gg/5TQTfSnSNp
  • Email: iqballoww@gmail.com
